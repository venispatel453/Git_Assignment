1.Command to initialize git repository:
git init


2.Command to clone remote repository:
git clone <repository_url>


3.Command to stage all changes:
git add .


4.Command to view the state of the working directory and staging area:
git status


5.Command to create a new branch:
git branch <branch_name>


6.Command to change branch:
git checkout <branch_name>


7.Command to clean the staging area:
git reset


8.Command to get commits and changes from a remote branch (without merging changes into the current branch):
git fetch origin <remote_branch_name>


9.Command to get commits and changes from a remote branch (merging changes into the current branch):
git pull origin <remote_branch_name>


10.Command to cherry-pick a commit with SHA 092018283103810930:
git cherry-pick 092018283103810930


11.Command to push a branch:
git push origin <branch_name>


12.Command to delete a remote branch:
git push origin --delete <branch_name>


13.Command to do iterative rebase:
git rebase -i HEAD~n
